React Shuffle
===

An example app to display & shuffle cards, made entirely in React & CSS.

Demo: https://react-shuffle.netlify.app

Click on cards to view.

### Setup

Requires [npm](https://npmjs.com).

Install dependencies:
```sh
npm install
```

Start dev server:
```sh
npm dev
```

Prod build:
```sh
npm build
```

React Shuffle
===

An example app to display & shuffle cards, made entirely in React & CSS.

Demo: https://react-shuffle.netlify.app

Click on cards to view.

### Setup

Requires [yarn](https://yarnpkg.com).

Install dependencies:
```sh
yarn install
```

Start dev server:
```sh
yarn dev
```

Prod build:
```sh
yarn build
```
